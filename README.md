
# Bintix Assignment

It is an authentication application using JWT token. 


## Tech Stack Used

 - Frontend
   - ReactJS
   - HTML5
   - CSS3
   - Material UI
 - Backend
   - MongoDB
   - express
   - Node

## Instructions to follow in CLI 

 - Backend
   - npm install
   - npm start
 - Frontend
   - npm install
   - npm start
